# WIP

Uma biblioteca simples em python implementando diversas utilidades básicas relacionadas a grafos, permitindo criar de grafos como objetos, requisitar informações sobre eles, e rodar algoritmos como Bellman-Ford e Dijkstra.

Criado como parte de um projeto de faculdade, na Universidade Estadual do Ceará.

# Nota temporária

Para criar funções, é só adicioná-las aos arquivos classes.py (para classes e funcionalidades báscias) ou algorimos.py (para outros algoritmos).

Para usar a biblioteca do jeito que está aqui, baixe, rode `python setup.py sdist bdist_wheel` na pasta, e então rode no local que você deseja instalar a biblioteca `pip install YOUR-PATH/trabalhografoslib/dist/trabalhografoslib-0.0.1-py3-none-any.whl`
